# Railway Ruby Sinatra Postgres

Go to <https://railway.app/new>.
Click "Deploy from Repo".
Select this repo.
Select `main` branch.
Click "Deploy".

It will fail.

Click "+ New" button.
Click "Database".
Click "Add PostgreSQL".

Click the original web service.
Click "Settings".
Set "Root directory" to `/railway-ruby-sinatra-pg`.
Set "Healthcheck Path" to `/`.
