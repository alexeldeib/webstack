webstack

A project to prototype web stacks: github.com/croaky/webstack

For each stack, SELECT 1 to a SQL database and return a JSON payload of { "status": "ok" }.

Runtimes / frameworks: Crystal / Kemal, Deno / stdlib, Go / stdlib, Node / Express, and Ruby / Sinatra.

Hosting platforms: Deno Deploy (TODO), Fly.io, Heroku, Netlify (TODO). Railway, Render, and Vercel.
