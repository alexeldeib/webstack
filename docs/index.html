webstack

A project to prototype programming languages and hosting platforms. Source code: croaky/webstack. I'm interested in experiencing each language and hosting platform.

I'd like to try a matrix of languages that SELECT 1 to a SQL database (mainly Postgres) using a popular database driver (no ORM), returning a JSON payload of { "status": "ok" }, deployed to modern hosting platforms.

The micro-frameworks include Sinatra (Ruby), Gin (Go), Express (Node), Oak (Deno), and Kemal (Crystal).

The modern hosting platforms include Deno Deploy, Fly, Netlify. Railway, Render, and Vercel.

In addition to experience, I'm interested latencies on each platform. The dashboard below should track latencies in real-time.
