webstack

github.com/croaky/webstack

A project to prototype web stacks: languages, frameworks, databases, and hosting platforms.

For each stack, SELECT 1 to a SQL database using a database driver (no ORM), and return a JSON payload of { "status": "ok" }, deployed to a modern hosting platform.

Fameworks / languages: Sinatra (Ruby), Gin (Go), Express (Node), Oak (Deno), and Kemal (Crystal).

Hosting platforms: Deno Deploy, Fly, Netlify. Railway, Render, and Vercel.
