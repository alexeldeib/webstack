webstack

github.com/croaky/webstack A project to prototype web stacks

For each stack, SELECT 1 to a Postgres database using a database driver (no ORM) and return a JSON payload of { "status": "ok" }.

Runtimes / frameworks: Crystal / Kemal, Deno / stdlib, Go / Gin, Node / Express, and Ruby / Sinatra.

Hosting platforms: Deno Deploy, Heroku, Netlify. Railway, Render, and Vercel.

The dashboard below shows real-time latencies from N. California, Montreal, Ireland, Stockholm, and Sydney.
